               ELECTRONIC LIBRARY SERVER.
==================================================
On this server, it is possible to issue books to 
users and release these books. There are 3 roles 
(person, librarian, admin).
The stack of technologies used:
- Java 1.8.0_251
- Maven
- Spring JPA(Hibernate)
- Spring MVC
- Spring Security
- Spring Test
- ThyMeLeaf
- JUnit 5
- SLF4J
- MySql
- Lombok
